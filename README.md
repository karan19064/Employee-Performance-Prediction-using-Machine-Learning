# Employee Performance Prediction using Machine Learning

## Overview
This project predicts employee performance using Machine Learning and HR analytics data. It helps identify factors influencing employee productivity and performance.

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Performance Prediction

## Algorithms Used
- Logistic Regression
- Random Forest
- XGBoost

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Workflow
1. Data Collection  
2. Data Cleaning  
3. EDA  
4. Model Building  
5. Prediction & Evaluation

pip install -r requirements.txt
